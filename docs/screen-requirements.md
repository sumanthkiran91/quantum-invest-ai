# Screen Requirements

## Shared navigation

- Dashboard
- AI Discover
- Smart Alerts
- Practice Invest
- Compare Brokers
- AI Daily Report
- Global search
- Notifications
- User profile
- Free or Premium badge
- Developer account switcher where testing requires it

## Global Dashboard

- Industry exploration must appear first and prominently.
- Market overview is compact and horizontal.
- Right column includes market timings for USA, Australia, India, UK, Europe, Japan, and crypto.
- Top Movers shows Top 5 Growing and Top 5 Losing and updates from industry selection.
- Additional cards include watchlist preview, portfolio summary, AI suggestions, news and alerts, daily report, practice investing, and Premium upgrade.

## Watchlist Dashboard

- Uses the same navigation as Global Dashboard.
- Displays only user-selected investments.
- Tabs: All, Shares, Crypto, ETFs, Commodities, Long-Term Picks.
- Includes search, sorting, filtering, add/remove, local storage, loading, empty, and error states.
- Rows open the reusable investment details page.

## Investment Details

- Works for any stock, cryptocurrency, ETF, or commodity.
- Includes asset header, chart filters, AI insight, scenario planner, news, key data, events, related investments, broker preview, Beginner/Expert modes, and Free/Premium states.
- Scenario planner uses deterministic mock outcomes by asset risk and historical profile.

## Feedback and admin viewer

- Feedback button appears on every page.
- Form captures page, clarity, liked feature, confusion, additions, removals, failures, willingness to pay, comments, issue reports, and improvement suggestions.
- Store locally and show tester consent, demo-data, and privacy notices.
