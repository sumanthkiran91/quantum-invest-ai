# User Flows

## First visit

1. User lands on registration or login.
2. User chooses an onboarding option: new to investing, already invest, or just exploring.
3. User may opt into the 7-day Practice Investing journey.
4. User enters the prototype app with a demonstration-data notice.

## Global market exploration

1. User opens Dashboard.
2. User sees "Explore Markets by Industry" first.
3. User selects an industry card.
4. Top Movers updates for the selected industry.
5. User opens an investment details page from a mover.

## Watchlist management

1. User opens Watchlist.
2. User filters by asset type or searches.
3. User adds or removes demo assets.
4. App persists watchlist locally.
5. Free users see a five-item limit; Premium users see unlimited access.

## Investment details

1. User opens any asset details page.
2. User reviews demo price, market status, trend filters, and AI insight.
3. User enters an amount in the scenario planner.
4. User compares short, medium, and long-term educational outcomes.
5. User may add the asset to watchlist or practice with virtual money.

## Feedback

1. User selects Feedback on any page.
2. User completes clarity, feature, issue, and willingness-to-pay questions.
3. Feedback is stored locally for prototype review.
4. Admin-style viewer displays submitted feedback.
